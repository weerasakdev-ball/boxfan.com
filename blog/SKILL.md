# 🥊 คู่มือการเขียนและส่งงานบทความ Boxfan

> **เป้าหมาย:** เพื่อให้การอัปเดตบทความขึ้นเว็บไซต์รวดเร็ว สวยงาม และไม่เกิด Error 
> รบกวนทีมเขียนทุกคนอ่านคู่มือนี้ และส่งงานมาให้ครบ **2 ส่วน** ในทุกๆ บทความนะครับ

## 📂 ส่วนที่ 1: การเขียนลงไฟล์หน้าบทความ (HTML)

เราจะใช้ไฟล์ต้นแบบ (`TEMPLATE.html`) ในการเขียนบทความเสมอ เพื่อให้โครงสร้างเว็บเหมือนกันทุกหน้า

**ขั้นตอนการทำ:**
1. เปิดไฟล์ `TEMPLATE.html` ด้วยโปรแกรมแก้ไขโค้ด (แนะนำ VS Code หรือ Notepad++)
2. พิมพ์เนื้อหาลงไป โดย **แก้ไขเฉพาะจุดที่มีเครื่องหมาย `[วงเล็บแบบนี้]` เท่านั้น** 
   * ⚠️ *ข้อควรระวัง: ระวังอย่าเผลอลบเครื่องหมายปีกกา `< >` หรือเครื่องหมายฟันหนู `" "` ของโค้ดหลักเด็ดขาด*
3. **รูปภาพหน้าปก (สำคัญมาก):** นำลิงก์ URL ของรูปภาพไปใส่แทนคำว่า `[URLรูปหน้าปก]` ซึ่งจะมีให้แก้ 3 จุด (อยู่ด้านบนสุด 2 จุด และตรง `<img class="art-hero"...>` 1 จุด)
4. **เนื้อหา:** พิมพ์แทรกระหว่าง `<p>` และ `</p>` หากต้องการขึ้นหัวข้อใหม่ให้ใช้ `<h2>` (อย่าลืมแก้สารบัญด้านบนให้ตรงกับชื่อหัวข้อ `<h2>` ด้วย)
5. **การเซฟไฟล์:** เมื่อเขียนเสร็จ ให้ **Save As** โดยตั้งชื่อไฟล์ตามกฎนี้:
   * ✅ ใช้ภาษาอังกฤษตัวพิมพ์เล็กทั้งหมด
   * ✅ ใช้ขีดกลาง `-` แทนเว้นวรรค
   * ✅ *ตัวอย่างที่ถูกต้อง:* `mma-advanced-techniques.html`
   * ❌ *ตัวอย่างที่ผิด:* `มวยไทย.html` หรือ `mma techniques.html`
